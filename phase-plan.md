# 基本設計〜Webサービス公開までのフェーズ分割

## フェーズ0：企画確定・要件凍結
**目的**
- 仕様ブレを止め、MVPの境界を明確にする。

**主な作業**
- コンセプト・対象年齢・価値仮説の確定
- MVP機能の確定（タップ/コンボ/成長/分岐進化）
- 非機能要件の初期定義（性能・セキュリティ・運用）
- 成功指標（KPI）設定

**成果物**
- 企画書 v1
- 要件定義書 v1
- KPI定義書

## フェーズ1：基本設計（システム/機能）
**目的**
- 作るものを設計図として言語化する。

**主な作業**
- 画面一覧・画面遷移設計
- ドメイン設計（セッション、タップイベント、モンスター状態）
- ポイント/コンボ計算ルール設計
- 進化分岐テーブル設計
- API基本設計（エンドポイント、入出力）
- データベース基本設計（ER図）
- 認証方式・アカウント方針（子ども/保護者）

**成果物**
- 基本設計書
- 画面遷移図
- API設計書（基本）
- ER図

## フェーズ2：詳細設計（実装可能レベル）
**目的**
- 開発チームが迷わず実装できる粒度に落とし込む。

**主な作業**
- 画面コンポーネント設計（状態、バリデーション）
- API詳細設計（エラーコード、再送制御）
- DB詳細設計（インデックス、制約、監査項目）
- コンボ判定ロジック詳細（タイムウィンドウ、上限）
- 不正/空タップ抑止ロジック詳細
- ログ設計（分析イベント定義）
- テスト設計（単体/結合/E2E観点）

**成果物**
- 詳細設計書
- OpenAPI定義
- テスト設計書
- 計測イベント仕様書

## フェーズ3：プロトタイプ開発（検証版）
**目的**
- 最小機能で体験検証を行い、早期フィードバックを得る。

**主な作業**
- 食事中画面の実装（タップ・コンボ・演出）
- 最小バックエンド実装（セッション保存）
- 最低限の保護者画面（履歴）
- ダミーデータで成長/進化検証
- UX簡易テスト（社内/身内）

**成果物**
- 動作プロトタイプ
- UX検証レポート
- 改善バックログ

## フェーズ4：本開発（MVP実装）
**目的**
- 公開可能品質まで機能を完成させる。

**主な作業**
- フロントエンド実装完成（全MVP画面）
- バックエンド実装完成（API/DB/認証）
- ポイント計算・進化ロジック本実装
- ログ収集・分析基盤接続
- 管理画面/運用補助機能（最低限）
- 文言・イラスト・サウンド反映

**成果物**
- MVP機能一式
- 運用管理機能（最低限）
- 分析計測実装済み環境

## フェーズ5：品質保証・セキュリティ・法務対応
**目的**
- 安全に公開できる状態を作る。

**主な作業**
- 単体/結合/E2Eテスト
- 負荷試験（ピーク時想定）
- 脆弱性検査（依存ライブラリ含む）
- プライバシーポリシー/利用規約整備
- 子ども向けサービスとしての同意導線確認
- 障害時運用手順（Runbook）作成

**成果物**
- テスト報告書
- セキュリティチェック報告
- 規約・ポリシー文書
- 運用手順書

## フェーズ6：公開準備（リリース運用）
**目的**
- 公開後も回る運用体制を整える。

**主な作業**
- 本番インフラ構築（監視/バックアップ/アラート）
- CI/CD最終整備
- Web公開導線整備（PWA設定を含む場合は対応）
- 初期ユーザー導線（オンボーディング）
- ロールバック手順確認
- リリース判定会

**成果物**
- 本番環境
- 監視ダッシュボード
- リリースチェックリスト
- Go/No-Go記録

## フェーズ7：段階公開（ソフトローンチ）
**目的**
- 小規模公開で利用実態を学び、改善する。

**主な作業**
- 限定ユーザー公開（クローズド）
- KPIモニタリング（継続率、連打率、セッション完了率）
- 問い合わせ/不具合対応
- バランス調整（コンボ窓、成長速度）
- A/Bテスト（演出・導線）

**成果物**
- ソフトローンチレポート
- 改善パッチ
- 正式公開判定資料

## フェーズ8：正式公開（一般公開）
**目的**
- 安定運用しながら成長サイクルへ移行する。

**主な作業**
- 一般公開
- 監視強化（初動72時間）
- KPI週次レビュー
- 優先度高い改善の短サイクル反映
- 次期機能計画（イベント、兄弟アカウント等）

**成果物**
- 正式リリース
- 運用週報テンプレート
- 次期ロードマップ

## MVPのスケジュール目安
- フェーズ0〜2：3〜5週間
- フェーズ3〜4：4〜8週間
- フェーズ5〜6：2〜4週間
- フェーズ7：2〜4週間

合計目安：11〜21週間（チーム規模・体制で変動）
