# Gemini Image Prompt

Create a polished game asset sheet for a children's web game called MoguMoguMonster.

Output requirements:
- One image containing 4 cute original baby monster characters arranged in a clean 2x2 grid
- Transparent or plain very light background
- Front-facing full body characters
- Consistent art style across all 4 characters
- Bright, friendly, premium mobile game illustration style
- Soft shading, clean silhouette, readable at small size
- No text, no UI, no watermark

Character directions:
- `momo`: pink mochi-like bunny monster, round cheeks, tiny leaf tail
- `pafu`: mint-green fluffy sheep monster, cloud-like wool, tiny golden bell
- `gabu`: orange mini dinosaur monster, little fang, playful expression
- `fuwa`: pale blue floating spirit monster, star motif, dreamy expression

Style constraints:
- Designed for children ages 3 to 7
- Very cute, safe, cheerful, non-scary
- Japanese casual game aesthetic
- Each character should feel distinct and memorable
- Keep poses simple so we can later derive idle, eat, and evolve variations

Framing:
- Leave generous padding around each character
- Keep all characters fully visible
- Use square composition suitable for later cropping into individual portraits
