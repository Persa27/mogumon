# 進化パターン表 v4.3（ブラッシュアップ版）

## 1. 目的

* 全ユーザー共通の `赤ちゃん` から始まり、食事傾向に応じて `子供 → 成熟体` と分岐し、最終的に `完全体` へと至る3段階の育成ツリーを定義する。
* 食事による分岐判定は「成熟体（第2段階）」で完了し、決定した25パターンの個性が、レベルアップによって「完全体（第3段階）」へと覚醒・巨大化する。
* **デザイン方針：** 「ポケットモンスター」シリーズのように、太めの主線、特徴的で感情豊かな瞳、シンプルな陰影を用い、**全段階を通して「親しみやすさ」と「相棒としての愛着」を持てるデザイン**で統一する。

## 6. キャラクター図鑑（詳細デザイン指示）

### 6-1. 子供（段階1：5パターン）

無垢で愛くるしく、プレイヤーが「守ってあげたい」と思えるデザイン。

1. **ドラップ (Draup)** [りんご系]: 赤い二足歩行のチビ竜。大きめのパッチリした瞳と、少し不器用そうに歩く姿が特徴。尻尾の小さな炎が感情で揺れる。
2. **ガロロン (Garoron)** [にんじん系]: オレンジの小狐。耳が長く、静電気で毛が常にふわふわしている。元気いっぱいでイタズラ好きな表情。
3. **フィシュル (Fishul)** [おさかな系]: 青く半透明な浮遊魚。水のバリアを纏っており、大きな水生生物特有の涼しげで澄んだ瞳を持つ。
4. **コロロン (Kororon)** [ブロコ系]: 緑の苔が生えた丸石のゴーレム。短い手足と、常にマイペースで穏やかなニコニコ顔。頭に小さな新芽。
5. **ルミナ (Lumina)** [バランス系]: 真っ白な体に虹色の瞳を持つ妖精。小さな星のようにフワフワと浮遊し、好奇心旺盛な仕草を見せる。

## 7. 実装・ID管理、及びグラフィック指針メモ

* **ID体系**: `[段階]_[進化1属性]_[進化2属性]`
* `0_punyun`
* `1_apple` (ドラップ)
* `2_apple_carrot` (ボルトドラグ)
* `3_apple_carrot` (インドラ・ドラグ)


* **画像制作・UI演出方針**:
* **主線と色彩:** ポケモンらしさを担保するため、線画はアウトラインを太めにし、アニメ塗りに近いスッキリした陰影（ベタ塗り＋1影＋ハイライト程度）にする。
* **目の表現:** クリーチャーになりすぎないよう、白目と黒目（またはハイライト）をはっきりと描き、感情移入できる「キャラクター」として扱う。
* **進化のコントラスト:** * `成熟体`は画面中央にスッキリと収まるアクティブなポージング。
* `完全体`はシルエットを大幅に巨大化（画面枠からはみ出るような構図や、パースを効かせたアングル）。炎、雷、光輪などのエフェクトパーツの密度を成熟体の3倍にし、画面全体を彩る「神々しさ」を演出する。
