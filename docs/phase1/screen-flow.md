# 画面一覧・画面遷移（フェーズ1）

## 1. 画面一覧
1. ホーム
2. 食事中
3. エサ種別選択
4. セッション結果
5. 履歴（保護者）
6. 初回オンボーディング

## 2. 主要遷移
- ホーム → 食事中（セッション開始）
- 食事中 → エサ種別選択（タップ時）
- エサ種別選択 → 食事中（確定後）
- 食事中 → セッション結果（終了時）
- セッション結果 → ホーム
- ホーム → 履歴

## 3. 画面責務
### ホーム
- 現在のモンスター状態表示
- セッション開始導線

### 食事中
- 大型タップボタン
- コンボ状態表示
- 進捗演出表示

### エサ種別選択
- 4〜5カテゴリの選択UI
- 選択確定後にポイント計算

### セッション結果
- 獲得ポイント
- 最大コンボ
- 成長/進化進捗

### 履歴
- 日次セッション結果一覧
